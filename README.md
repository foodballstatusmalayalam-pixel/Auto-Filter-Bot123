**Trinity AutoFilter** by **Trinity Mods** turns your channels into an instantly searchable
movie / series library — with a built-in **streaming engine** that converts any file into a
seekable watch link or direct download on the fly, nothing buffered to disk.

> 🌀 **Best-in-class performance, cleanly organized codebase.**
> 🔧 **Tune it from `config.py` / `sample.env` — no code edits required.**
> 🚀 **Deploy in under 2 minutes on Koyeb / Render / Railway / Docker / VPS.**

```
Auto-Filter-Bot/
```

<p align="center">
  <img src="https://capsule-render.vercel.app/api?type=soft&color=gradient&customColorList=20,12,6&height=58&section=header&text=%E2%9A%99%EF%B8%8F%20Environment%20Variables&fontSize=26&fontColor=ffffff&animation=twinkling&fontAlignY=58" width="70%"/>

> Full, commented list lives in [`sample.env`](sample.env) and [`config.py`](config.py).

| Variable | Description |
|---|---|
| `API_ID` · `API_HASH` | From https://my.telegram.org/apps 🔴 |
| `BOT_TOKEN` | From [@BotFather](https://t.me/BotFather) 🔴 |
| `ADMINS` | Admin user id(s), space-separated 🔴 |
| `DATABASE_URI` · `DATABASE_NAME` | MongoDB (Atlas) URI + db name 🔴 |
| `LOG_CHANNEL` | Log channel id `-100…` (bot must be admin) 🔴 |
| `CHANNELS` | Channel(s) to auto-index |
| `BIN_CHANNEL` · `TMSL` · `PORT` | Streaming: storage channel · **Trinity Mods Streaming Link** (public domain) · port |
| `AUTH_CHANNEL` | Force-sub channel id(s) (one or many) |
| `MULTI_TOKEN1…N` | Extra worker bot tokens for faster streaming |
| `IS_VERIFY` · `VERIFY_URL` · `VERIFY_API` | Shortener-verification gate |
| `AUTO_FILE_DELETE` · `AUTO_DELETE_TIME` | Auto-delete delivered files |
| `LIMIT_MODE` · `DAILY_FILE_LIMIT` · `DAILY_SENDALL_LIMIT` | Free-user daily limits |
| `INSIGHTS_MODE` | Toggle search analytics (`/trending`) |
